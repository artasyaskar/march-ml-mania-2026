# Competition Data

This folder should contain all CSV files required for the **March Machine Learning Mania 2026** competition.

> **Important:** The data files are **not included** in this repository due to GitHub size limits.  

## How to get the data

1. Go to the Kaggle competition page:  
   [March Machine Learning Mania 2026 Data](https://www.kaggle.com/competitions/march-machine-learning-mania-2026/data)

2. Download all the CSV files (either individually or as a zip).

3. Extract the zip (if downloaded as zip) and copy **all CSV files** into this folder (`Competition_data/*.csv (files)`).

After placing the files here, you can run the project code and notebooks as intended.
